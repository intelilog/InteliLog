# Objetivo do Projeto  
Este projeto tem como objetivo desenvolver um dashbord, que permita:
* Analisar indicadores de segurança viária: Avaliar métricas como mortalidade, taxa de sinistros, gravidade dos acidentes, frota e população, com foco em veículos pesados.
* Realizar comparações regionais: Permitir a análise comparativa entre estados e regiões do Brasil, identificando padrões e variações nos índices de sinistralidade.
* Avaliar a evolução temporal: Analisar o comportamento dos indicadores de segurança viária ao longo do tempo, identificando tendências de crescimento ou redução.
* Analisar fatores de risco: Identificar possíveis relações entre variáveis, como o crescimento da frota de veículos pesados e o aumento de acidentes.
* Classificar municípios por desempenho: Elaborar um ranking de eficiência em segurança viária para municípios com mais de 250 mil habitantes.
